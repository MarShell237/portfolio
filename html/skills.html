  1. Programming Languages:

        Framework:

      • JavaFX
        • JavaFX is a Java library that allows you to create rich graphical applications. It is used to create user interfaces for desktop and mobile applications.
    • JavaScript
      • JavaScript is a programming language that allows you to add interactivity to a web page. It is used to create animations, games, and web applications.
  2. UI / UX Design:
    • Scene Builder
      • Scene Builder is a tool for creating graphical user interfaces for JavaFX applications. It allows you to create user interfaces visually, without having to write code.
  3. Web Development:
    • HTML
      • HTML (HyperText Markup Language) is a markup language used for creating the structure of a web page. It is used to define the different elements of a page, such as the title, paragraphs, images, and links.
    • CSS
      • CSS (Cascading Style Sheets) is a programming language used to style a web page. It is used to control the color, font, size, and layout of the elements of a page.
  4. Database Technologies:
    • MySQL
      • MySQL is an open-source relational database management system (RDBMS). It is used to store and manage data for web applications and desktop applications.
    • SQLite
      • SQLite is an embedded RDBMS that is used to store data in files. It is often used for mobile applications and embedded applications.
